用JavaURL编程爬取并分析网页敏感词
实践学时：16
实践目的或任务：通过指导学生上机实践，对JavaURL编程、Java图形界面进行掌握。
实践基本要求：
1.了解实验目的及实验原理；
2.编写程序，并附上程序代码和结果图；
3.总结在编程过程中遇到的问题、解决办法和收获。
实践的内容或要求：
1.编写界面，输入一个网址，能够爬取该网址上所有的HTML源代码。
2.对网址中的文本进行提取。
3.建立敏感词库，用文本文件保存。
4.将该网址所对应的文本中的敏感词提取并高亮显示。
5.编写文本文件，可以存入多个网址；程序可爬取这些网址中的文本内容，将敏感词记录存入另一个文件，格式自定。
